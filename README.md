# Cortex — Upcycled Hexagonal Hanging Planters

A single-page, GitHub Pages-ready website for Cortex — an upcycling initiative
that converts discarded PET water bottles into 3D-printed modular hanging
planters with gravity-fed wooden water reservoirs.

> **Built from waste. Engineered for growth.**

## What's inside

- **`index.html`** — Full multi-section site (hero slideshow, mission, origin story, problem stats, material science, product render, original sketches, supply chain, unit economics, impact model, team)

## Product concept

- **Material:** Recycled PETG (rPETG) extruded from post-consumer PET water bottles
- **Structure:** 5-cell hexagonal cluster, 3D-printed as one piece
- **Hydration:** Wooden pine reservoir → metal straw → gravity-drips to top cell → cascades via drainage channels
- **Base:** 304 stainless steel waterproof cover
- **Hook:** Integrated loop on feed straw
